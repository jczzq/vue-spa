# brand-center

> vue 单页应用 项目模板
集成了
1. iconfont
2. axios
3. 基础less
4. 基础路由、指令及过滤器
5. 基础postcss插件（px2rem：移动端适配）
6. 项目分层，结构清晰，易于拓展

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
